# MCP Scanner MVP Implementation Plan

> **For agentic workers:** Use superpowers:subagent-driven-development to implement this plan.

**Goal:** 构建 MCP 暴露面扫描器 MVP，输入 IP/CIDR/域名，输出暴露的 MCP 服务及风险等级。

**Architecture:** 五阶段串行流水线：目标解析 → TCP 端口扫描 → HTTP 筛选 → MCP 指纹识别（三层打分≥0.65）→ 工具枚举+风险评级+蜜罐检测。全程 Go 原生 net/http + goroutine。

**Tech Stack:** Go 1.21+, urfave/cli/v2, net/http, golang.org/x/sync/semaphore
